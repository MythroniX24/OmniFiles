# Changelog

## [1.0.0] - 2026-07-20

### Added
- Initial release of OmniFiles
- File browsing with Material 3 UI
- File operations: copy, move, rename, delete, share, create
- Indexed search engine with name, extension, date, and size filters
- Archive support: ZIP, 7Z, TAR, GZIP (compress and extract)
- Storage support: Internal, SD Card, USB OTG
- Theme system: Light, Dark, System, Dynamic Colors
- Quick access grid on home screen
- Favorites management
- Recent files tracking
- File sorting by name, date, size, extension
- Settings page with theme and file browsing options
- Smooth animations and transitions
- Responsive layout for phones and tablets
- 100% offline, no ads, no tracking, no analytics
