# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability, please report it privately.

**Do not report security vulnerabilities through public GitHub issues.**

Instead, please email the project maintainer directly.

## Security Features

- **100% offline operation** — No data leaves your device
- **No analytics** — We never track usage
- **No ads** — No third-party ad SDKs
- **No unnecessary permissions** — Only storage permission is required
- **Minimal dependencies** — Reduced attack surface
